项目经历

  • 途咖租车移动互联网应用及平台 2012/12-2014/3 (15 个月)
    担任角色： Product Manager/Architect

    此产品是一个租车的移动互联网应用。提供App、微信、机顶盒、Web等多种途径订车，由自动调度服务器调度整个业务流程，并提供业务支撑及管理平。
在此项目中，主要承担任务：1）完成整体的技术架构；2）完成产品设计、验收。

  • 农产品收购系统 2011/6-2012/8 (14 个月)
    担任角色： Architect

    此系统是对益海嘉里全国各地工厂的农产品收购业务进行全流程的监控与管理。系统采用Java EE 6技术体系实现，采用IBM的Websphere/DB2集群做为后台支撑，采用Message中间件、流程引擎进行业务解耦，是个大型的企业应用。
在此项目中，主要承担任务：1）完成系统整体的技术架构；2）完成系统的业务分析及设计；3）制定部门的敏捷研发流程。

  • 健康检查系统 2010/8-2011/4 (6 个月)
    担任角色： Business Analyst/Design

    此项目是一家商业体检中心的业务系统，它包括公司内部业务管理、体检管理、CRM等。本人在此项目中担任主要的需求分析和设计工作，本项目采用敏捷建模（Agile Modeling）和RUP结合的方式进行需求分析和设计。
 在此项目中，主要承担任务：1）完成技术方案和招标；2）完成需求分析及设计。

  • 电子政务系统 2009/12-2010/6 (7 个月)
    担任角色： Project Manager/Architect

    此项目是为成都市及下级县市所有行政权力事项（行政许可、行政处罚、行政强制、行政征收、其他行政权力等）实现在线统一接件和办理系统。此项目采用了基于SSH2的架构实现、基于RUP的分析和设计方法和Scrum的敏捷开发模式。
在此项目中，主要承担任务：1）完成需求调研及分析；2）完成系统架构与设计；3）管理整个项目至交付。

  • 重庆移动经营分析系统 2004/9-2005/5 (8 个月)
    担任角色： Software Engineer

    此项目亚信公司为重庆移动开发的一个经营分析系统。系统对数据进行ETL、清洗后，装入DB2和Oracle等数据库，采用Perl/Python/C等语言对数据进行分析，将分析结果展现到前台。系统基于Unix进行开发。
在此项目中，主要承担任务：1）经营分析系统的维护及部分开发。

  • DIR 项目（电信） 2003/10-2004/7 (9 个月)
    担任角色： Project Manager
    此项目是和日本大和电信一个长期外包合作项目。系统基于 Unix 大型机开发，前端采用 JSP/Servlet 来展现。
    在此项目中，主要承担任务：1）项目管理。

  • TME（ERP） 2002/6-2003/9 (15 个月)
    担任角色： Project Manager
    此项目是日本 NEC 公司的一个企业 ERP 系统。系统基于 J2EE 的 B/S 三层架构，前台采用 JSP、Swing 和 Applet，后台采用 EJB2 实现。此项目人数达 25 人。
    在此项目中，主要承担任务：1）项目开发过程管理；2）解决重大技术问题。
