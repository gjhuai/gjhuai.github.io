改变docker的image存放目录

steps:

  1. mkdir /data/docker（你想要docker存放image的目录）
  2. systemctl stop docker
  3. vi /usr/lib/systemd/system/docker.service
  4. 添加 -g /data/docker
  5. cp -R /var/lib/docker/* /data/docker/
  6. systemctl daemon-reload
  7. systemctl start docker
