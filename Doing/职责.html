如何学习新技术？

  • 快速了解：看视频，入门 blog，专题 blog。
  • 解读代码：去搞清楚每一行代码，并笔记记录。
  • 代码选择：github 上的 code tutrial 库最佳，零散代码亦可。
  • 不是为了保留代码，是为了记住、理解代码。
  • code tutorial 库可长期维护，温故知新。
  • 系统学习：看书，做笔记。
  • 在掌握技术的用法后，看一本书（不要太多的书）系统的梳理知识点。
